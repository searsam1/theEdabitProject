Create a left rotation and a right rotation function that returns all the left rotations and right rotations of a string.
~Examples~
left_rotations("abc") ➞ ["abc", "bca", "cab"] right_rotations("abc") ➞ ["abc", "cab", "bca"] left_rotations("abcdef") ➞ ["abcdef", "bcdefa", "cdefab", "defabc", "efabcd", "fabcde"] right_rotations("abcdef") ➞ ["abcdef", "fabcde", "efabcd", "defabc", "cdefab", "bcdefa"]
